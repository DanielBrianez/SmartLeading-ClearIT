# 🗺️ Roadmap do Produto - Smart Leading V2 (Clear IT)

Este documento traça a visão de evolução do produto, desde a fundação (MVP) até à sua integração total no ecossistema corporativo da empresa sob o paradigma **Agent-First**.

## 🚀 Fase 1: MVP (Fundação e Validação) - *Concluído*
O objetivo desta fase foi validar o conceito principal, garantindo a geração de valor imediato com o menor esforço de desenvolvimento possível e máxima segurança.
- [x] Estruturação da arquitetura moderna de arquivos (*Clean Code* com React + Python/FastAPI).
- [x] Integração com a API do Google Gemini (Motor de Inteligência Artificial).
- [x] Criação do Mega-Prompt com foco na adaptação comportamental do líder e liderado (Metodologia CRIA).
- [x] Geração de Ata Oficial em PDF (*html2pdf.js*) com formatação visual avançada no front-end.
- [x] Conformidade total com a LGPD (*Privacy by Design* e **Semantic Firewall** Client-Side ativo mascarando dados financeiros e PII via RegEx).

## 🎨 Fase 2: UX Premium e Governança Bilateral (Aceleração) - *Concluído*
Foco na construção de uma interface imersiva, eliminação do microgerenciamento fantasma e recolha de telemetria baseada no navegador.
- [x] Interface de alto nível com *Glassmorphism* e Tailwind CSS (Suporte a *Dark Mode*, *Modals* animados e *Toasts* de notificação flutuantes).
- [x] Implementação da **Governança Bilateral de XP**: O XP gamificado da Liga de Ouro (Patentes de Bronze a Diamante) só é creditado mediante avaliação (Nota de 1 a 5 estrelas) do próprio Liderado.
- [x] Construção do **Playbook do Líder**: Central imersiva de microlearning (estilo *streaming*) fornecendo frameworks práticos (Feedback, PIP, Burnout) no exato momento da dor do gestor.
- [x] Implementação da **Trava Estratégica de Foco**: Limite estrutural de no máximo 3 PDIs ativos por pessoa (evitando *burnout*), enquanto missões táticas (Tasks) operam sem limites.
- [x] **Radar de Cadência e Cross-Routing**: Dashboard proativo com alertas de 1:1s atrasadas, adiamento inteligente (+7 dias) e auto-preenchimento de pautas.

## 📊 Fase 3: Portal de *People Analytics* (Painel de RH) - *Concluído*
Expansão do produto para atender aos executivos e gestores de Recursos Humanos, transformando o "Achismo" em dados organizacionais acionáveis.
- [x] Criação do Dashboard gerencial "Visão de Deus", cruzando os dados da telemetria da empresa inteira de forma descentralizada.
- [x] Implementação do **Termômetro de Risco Organizacional** para monitoramento preditivo e alertas de Sobrecarga e *Burnout*.
- [x] Identificação automatizada do *Compliance* de Cadência, mapeando "Lideranças Ofensoras" com maior volume de 1:1s em atraso.
- [x] **Geração em 1-clique do Relatório DRE** (Demonstrativo de Resultado de Engajamento) em formato PDF corporativo A4[cite: 1].
- [x] Sistema *Dev Tools* embutido para autoinjeção de base de dados para testes em demonstrações comerciais.

## 🔌 Fase 4: Cultura 360 e Integrações Nuvem (Escala) - *Futuro*
Avançar o produto para o ecossistema na Nuvem, integrando as ferramentas que as equipas já utilizam diariamente e descentralizando o reconhecimento.
- [ ] Implementação de Sistema de Kudos (Mural 360), permitindo gamificação horizontal e troca de XP entre membros de diferentes squads.
- [ ] Integração de notificações reais através de *Webhooks* para o Slack ou Microsoft Teams.
- [ ] Integração com a API do Google Calendar / Outlook para agendamento automático e disparos de invites com a pauta estruturada.
- [ ] Migração da base de dados local para uma arquitetura Cloud (ex.: Supabase, PostgreSQL) para consolidar a Liga de Ouro entre múltiplos dispositivos corporativos reais.
